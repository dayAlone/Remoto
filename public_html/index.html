technical specifications

technical specifications

device
  • Operational temperature: -40 – +125 C
  • Form-factor: external 3G/LTE+GNSS module and detachable OBD-connector
  • Size: L 85 x W 58 x H 30 mm
network
  • Integrated 3G/LTE Wi-Fi router
  • Supports GSM 900/1800 MHz, WCDMA, LTE-FDD, LTE-TDD bands
  • Wireless access point 802.11 b/g/n
  • Over-the-air firmware upgrades
extras
  • OBD connection with CAN bus & K-line monitoring
  • Built-in audio card with FM transmitter and AUX 2,5mm
  • USB connectors for additional peripherals
sensors
  • Integrated GPS/GLONASS engine and antenna
  • Built-in G-sensor (for motion, tilt and towing detection)
  • Temperature sensor
  • Onboard voltage monitoring
status LEDs
  • Device working mode
  • 3G/LTE network registration
  • Wireless AP status
Memory
  • 64 Mb embedded DDR2 RAM
  • 16 Mb embedded flash
  • Internal flash storage 32 Gb
